◆ recv()

bool BluetoothClient::recv ( void  )

Attempts to read a byte from the bluetooth connection until a full packet is recieved.

This must be called repeatedly until it returns true. Packet data will not start being stored until PACKET_START is read, at which point the buffer will begin to be populated. Once the full packet is recieved and the checksum is valid, this function will overwrite lastPacket (getLastPacket()) and return 1 ONCE. If you call it again, it will try looking for another packet BUT WILL NOT overwrite the last packet until a full new packet is found and validated.

Returns
1 if a full packet has been read and checked, 0 otherwise
